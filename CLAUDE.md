# ГРАФИН — QR-меню ресторана

Цифровое меню для ресторана (файн-дайнинг), открывается по QR-коду на столике.
**Mobile-only** веб-приложение. Стек: Next.js + Supabase (Postgres + Storage + Auth),
прод на Vercel. Десктоп-вёрстка не делается.

Дизайн-док проекта: `~/.gstack/projects/nnvrsln-AutoRent/Ruslan-main-design-20260606-193918.md`

## Design System
Перед любыми визуальными/UI-решениями читай `DESIGN.md`.
Все шрифты, цвета, отступы, эстетика и спецификация видео-фона определены там.
Не отклоняйся без явного согласия пользователя.
В режиме QA отмечай код, не соответствующий `DESIGN.md`.

Ключевое: только мобильная вёрстка; шрифты Playfair Display + Manrope (обе с кириллицей,
шрифты без кириллицы не использовать); бумажный фон меню `#EAE6DD`, тёмное видео-приветствие;
видео-фон в прод только как mp4+webm+poster (не `.MOV`).
